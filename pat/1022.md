# 1022. D进制的A+B(20)

- 时间限制：100 ms
- 内存限制：65536 kB
- 代码长度限制：8000 B
- 判题程序：Standard
- 作者：CHEN, Yue

输入两个非负10进制整数A和B(<=230-1)，输出A+B的D (1 < D <= 10)进制数。

**输入格式**

输入在一行中依次给出3个整数A、B和D。

**输出格式**

输出A+B的D进制数。

**输入样例**

```
123 456 8
```

**输出样例**

```
1103
```

----------

**编译器**

C (gcc 4.7.2)

**代码**

```c
#include <stdio.h>

int main(void) {
    int a = 0, b = 0, base = 0;
    scanf("%d %d %d", &a, &b, &base);

    int number = a + b;

    enum {MAX_D = 30};
    int digits[MAX_D];
    
    int i = 0;
    do {
        digits[i++] = number % base;
        number /= base;
    } while(number != 0);

    for (--i; i >= 0; i--) printf("%d", digits[i]);

    return 0;
}
```
